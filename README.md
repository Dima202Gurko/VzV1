# Консольное приложение для записи пользователя в БД

## Запуск программы
1. С репозитория **VzV1** необходимо скачать файл **Pr1.cs**
2. В своём компмиляторе создайте консольное приложение
3. Замените файл **Program.cs** на **Pr1.cs**
    - *Вы также просто можете скопировать код с **Pr1.cs** и заменить им код файла **Program.cs** *
4. Запустите код и выполняйте то, чего требует от вас консоль
